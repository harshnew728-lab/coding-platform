# Coding Platform

A coding platform that allows users to solve programming problems, submit solutions, and receive real-time evaluation using an external code execution API.

---

## 🚧 Project Status

This project is currently **under development**.

* Backend is largely implemented
* Core features like authentication, problem management, and code submission are functional
* Frontend is yet to be developed

---

## 🚀 Features

### 👤 User & Admin

* User registration & login
* Secure authentication
* Admin registration

### 🧩 Problem Management

* Create coding problems (Admin)
* Delete problems
* Manage problem data

### 💻 Code Submission

* Submit solutions in different languages
* Code execution using Judge0 API
* Token-based submission tracking
* Fetch and display execution results

---

## 🛠️ Tech Stack

**Backend:**

* Node.js
* Express.js
* MongoDB (Mongoose)

**Libraries & Tools:**

* bcrypt
* validator
* JWT (for authentication)
* Judge0 API (code execution)
